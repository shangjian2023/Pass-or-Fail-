# 期末求生计算器

一款专为中国大学生设计的期末成绩计算器，帮你精准计算需要考多少分才能达到目标成绩。

![期末求生计算器](https://img.shields.io/badge/期末求生-计算器-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 功能特点

### 核心计算功能
- **常规比例制计算**：根据平时分和权重计算期末所需分数
- **多场景目标设定**：不挂科(60分)、冲绩点(80分)、保研线(90分)
- **智能情绪反馈**：根据计算结果提供不同的表情和提示

### 交互设计
- **三步完成计算**：平时分 → 权重 → 目标分
- **快捷按钮**：常用权重(30%/40%/50%/60%)一键选择
- **滑块输入**：直观调整数值
- **实时同步**：数字输入与滑块双向绑定

### 情绪反馈系统
| 分数范围 | 表情 | 状态 | 提示 |
|---------|------|------|------|
| >100分 | 😱 | 难度爆表 | 难度较大，需超常发挥！ |
| 85-100分 | 😰 | 挑战较大 | 需要努力复习，加油！ |
| 60-85分 | 😊 | 压力适中 | 压力不大，但也要认真复习哦 |
| 0-60分 | 😄 | 轻松达成 | 恭喜！你已经稳过了 |
| <0分 | 🎉 | 稳过无疑 | 即使不考也能及格，太棒了！ |

### 分享功能
- 生成精美的成绩预测卡片
- 支持下载为PNG图片
- 包含祝福语："转发此图，期末必过！"

## 使用方法

### 在线使用
直接在浏览器中打开 `index.html` 文件即可使用。

### 本地启动
```bash
cd 期末求生计算器
python -m http.server 8080
```
然后访问 http://localhost:8080

### 计算步骤
1. **输入平时分**：输入你已获得的平时成绩（0-100分）
2. **选择平时分占比**：点击常用比例按钮或拖动滑块
3. **设定目标分数**：选择预设目标或输入自定义分数
4. **点击计算**：查看结果和情绪反馈

## 计算公式

```
期末需要分数 = (目标总分 × 100 - 平时成绩 × 平时权重) / (100 - 平时权重)
```

例如：
- 平时分：85分
- 平时权重：40%
- 目标总分：80分
- 期末需要考：76.7分
